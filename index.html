Introduction to VS code

Some information about Visual Studio Code : Visual Studio Code is a lightweight but powerful source code editor which runs on your desktop and is available for Windows, macOS and Linux. It comes with built-in support for JavaScript,TypeScript and Node.js has a rich ecosystem of extensions for other languages (such as C++, C#, Java, Python, PHP, Go)

Please Open Console

Some information about Console : A console traditionally refers to a computer terminal where a user may input commands and view output such as the results of inputted commands or status messages from the computer.
